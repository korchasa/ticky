# ticky
Tiny issue tracking in source code

[:ticky:]

## features

 - markdown-based issues 
 - `kanban board` in README.md (by pre-commit hook) 
 - custom issue statuses support 
 - cli tool with simple commands and autocomplete
    * `new` issue
    * `my` issues
    * `list` issues
        - `all` [default]      
        - by `assignee` 
        - by `status`
    * `status` change
